# pythonでprintの上書きをする方法
## 一行上書きする場合
一行上書きする場合、`\r`（キャリッジリターン）を使います。これはカーソルを行の先頭に戻す制御文字です。
```bash
print("\r"+"一行上書きするよ！",end="")
```
`end=""`を使って改行を行わないようにします。
## n行上書きする場合
n行上書きする場合、特殊なエスケープシーケンス`\033[nA`を使用します。ここで、`n`は上書きする行数です

```bash
print("n行上書きするよ！"+"\033[nA")
```
