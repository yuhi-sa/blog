# sshの設定
```bash
$ sudo apt-get install aptitude
$ sudo aptitude install ssh
```

/etc/ssh/sshd_configを編集
```bash
PermitRootLogin no
```
```bash
$ sudo /etc/init.d/ssh restart
```

# warningが発生した場合
```bash
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
@    WARNING: REMOTE HOST IDENTIFICATION HAS CHANGED!     @
@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@@
```
となった場合
```bash
ssh-keygen -R ipアドレス
```
