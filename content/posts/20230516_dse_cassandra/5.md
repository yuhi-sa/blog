Pythonドライバを使用してApache Cassandraに接続する。

```test.py
from cassandra.cluster import Cluster
cluster = Cluster(protocol_version = 3)
session = cluster.connect('killrvideo')

for val in session.execute("SELECT * FROM videos_by_tag"):
    print(val)
```
