クラスタリングカラムとは、主キーの一部であるが、パーティションキーではないカラムのこと。

## クラスタリングカラムを指定してテーブル作成
```
CREATE TABLE videos_by_tag (
    tag text,
    video_id uuid,
    added_date timestamp,
    title text,
    PRIMARY KEY ((tag), added_date, video_id)
    ) WITH CLUSTERING ORDER BY(added_date DESC);
```
